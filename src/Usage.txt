Scrapegis. Used for fetching property records from Hennepin County's GIS
server.

    $ scrapegis fetch pid 3302924110099
    3302924110099,2320,"COLFAX AVE S",55405,"M ... 

    $ scrapegis fetch names %BARRY%G%
    ...

    $ scrapegis fetch names "GARY S%"
    ...

Usage: 
  scrapegis fetch city [options]
  scrapegis fetch zip <zip_code> [options]
  scrapegis fetch taxpayer <name_like> [options]
  scrapegis fetch owner <name_like> [options]
  scrapegis fetch names <name_like> [options]
  scrapegis fetch pid <pid> [options]
  scrapegis query <query_string> [options]

Options:
  -m, --mock         Use mocked data
  -c, --csv          Output to CSV
  -j, --json         Output to simpler JSON
  --no-header        Do not include CSV header in output.

  --out=<outfile>    Output to file (stdout if not present)
